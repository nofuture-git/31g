Types of Tests


Automated Unit Test
A program, it being a test that is itself code. This form of tests is specific to Test-Driven Development and is typically facilitated by NUnit or MsUnit.
Manual Unit Test
A participle, an ordered list of steps taken by a developer whose conclusion is the confirmation of a code change functioning as intended.
Unit Test
A rubric, whose meaning may include automated, manual or an entry of a "Test Cases" within Rally.
QA Test
A participle, in which a trained professional performs event-driven actions upon a system in all or most substantial permutations.
Regression Test
A recurrance, being a series of other tests regarded as aspects to essential functionality.
Integration Test
An interdependence, whose concern is the overlapping aspects of individual changes.
End-to-End Test
An amalgamation, whose concern is the fitting of an internal change to its multifarious environment.
Performance Test
A ratio, being an action denominated in some measure of time (usually seconds).
Security Test
A rubric, whose form is in attempting any known vulnerabilities.
Penetration Testing
A participle, in which a professional attempts to gain access to a computer system that is supposed secure.
Code Coverage Test
A manifold, whose concern is the division of aspects between what has and what has not been tested.
Usability Test
A participle, specifically concerning the targeted end-users in which said users interact with the application to any measure of their own concern.
Stress Test
An investigation, whose intention is to delimit the upper limits of capacity with regards to its target.
Endurance Test
An investigation, whose intention is to delimit the amount of time a system may sustain a continuous expected load.
Spike Test
An observation, by which a system is suddenly demanded to service a vastly larger amount of request then it was asked to service only moments before.
Configuration Test
A contrast, which regards the outcome of a system under normal performance but which has had key configuration settings changed.
Isolation Test
A reduction, by which an action, series of actions, or recurrance of an action is performed with making smaller the manifold of possiabilities down to its lowest logical level.
